# Nodejs Full Stack Application

A Simple Node.js app using Express 4

Frontend - HTML, CSS, JS
Backend - Nodejs, Express

## Running Locally

```sh
$ npm install
$ npm start
```

Your app should now be running on [localhost:8000](http://localhost:8000/).
